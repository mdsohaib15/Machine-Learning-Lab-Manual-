## Overview

This repository contains implementations of all **14 Machine Learning laboratory experiments** prescribed in the official ML practical manual.
Each lab focuses on a core machine learning concept, starting from data preprocessing and classical supervised learning models to clustering, computer vision, AR-based applications, and a final complex project.

All labs are implemented in **Python** using standard ML and CV libraries, with proper evaluation and visualization where applicable.

## Lab Details

### LAB # 01 – Data Preprocessing

**Objective:**
Perform data cleaning, encoding of categorical features, and feature scaling to prepare raw data for machine learning models.

**Key Concepts:**
Missing values handling, label encoding, one-hot encoding, normalization, standardization.

---

### LAB # 02 – Simple & Multiple Linear Regression

**Objective:**
Implement simple and multiple linear regression models to analyze relationships between dependent and independent variables.

**Key Concepts:**
Regression line, coefficients, prediction, model evaluation.

---

### LAB # 03 – Logistic Regression (Binary & Multiclass)

**Objective:**
Apply logistic regression for classification problems involving binary and multiclass targets.

**Key Concepts:**
Sigmoid function, decision boundary, accuracy, confusion matrix.

---

### LAB # 04 – Decision Tree Classifier

**Objective:**
Build a decision tree model to classify data using rule-based splitting.

**Key Concepts:**
Entropy, information gain, tree depth, overfitting.

---

### LAB # 05 – Random Forest Classifier

**Objective:**
Improve classification performance using an ensemble of decision trees.

**Key Concepts:**
Bagging, ensemble learning, feature importance.

---

### LAB # 06 – Support Vector Machine (SVM)

**Objective:**
Classify data using Support Vector Machines with different kernels.

**Key Concepts:**
Hyperplanes, margin maximization, kernel trick.

---

### LAB # 07 – Open Ended Lab

**Project:** Student Performance Predictor
**Description:**
A supervised ML application that predicts whether a student will pass or fail based on study hours, attendance, and assignment scores.

**GitHub Repository:**
https://github.com/mdsohaib15/Open-Ended-Lab-Student-Performance-Predictor

---

### LAB # 08 – K-Nearest Neighbors (KNN)

**Objective:**
Implement KNN classifier and analyze model performance across multiple values of K.

**Key Concepts:**
Distance metrics, K selection, accuracy, confusion matrix.

---

### LAB # 09 – Naïve Bayes Classifier

**Objective:**
Apply probabilistic classification using Naïve Bayes assumptions.

**Key Concepts:**
Bayes theorem, conditional probability, independence assumption.

---

### LAB # 10 – K-Means Clustering

**Objective:**
Cluster unlabeled data into groups using K-Means clustering.

**Key Concepts:**
Centroids, inertia, elbow method, unsupervised learning.

---

### LAB # 11 – Hierarchical Agglomerative Clustering

**Objective:**
Perform hierarchical clustering and analyze data structure using dendrograms.

**Key Concepts:**
Linkage methods, dendrogram interpretation, cluster hierarchy.

---

### LAB # 12 – ML × AR: Real-Time Object Classification Overlay

**Objective:**
Integrate machine learning with real-time video input to classify objects and overlay predictions.

**Key Concepts:**
Computer vision, real-time inference, OpenCV integration.

---

### LAB # 13 – ML × AR: Gesture Recognition

**Objective:**
Develop a real-time hand gesture recognition system using hand landmarks.

**Gestures Implemented:**

- Open Hand
- Fist
- Pointing

**Approach:**
MediaPipe-based hand landmark detection with rule-based and ML-assisted classification.
